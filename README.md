# FitCircle

Plataforma web + app híbrida para deportistas. Centraliza la monitorización física, la comunidad y la localización de espacios deportivos, con un enfoque multiplataforma y de bajo coste.

## Qué incluye este prototipo

- Página de inicio con toda la información del proyecto.
- Secciones de funcionalidades, requisitos, viabilidad y presupuesto.
- Navegación inferior consistente entre páginas.
- Pantallas de rutas, retos y perfil ya disponibles.
